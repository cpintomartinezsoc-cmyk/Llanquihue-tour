📘 Descripción general del sistema

El objetivo de este proyecto es leer información de los tours almacenados en un archivo de la base de Llanquihue Tour (.txt), a partir de el, crear objetos y que datos queden almacenados en una colección dinámica (ArrayList).
Igualmente se permite recorrer el listado completa y filtrar según la necesidad del usuario, facilitando la gestión de información de los tours ofrecidos por la agencia.
